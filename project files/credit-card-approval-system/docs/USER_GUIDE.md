# User guide

1. Create an account, then sign in.
2. Choose **Apply**, complete the applicant profile, and submit it.
3. Review the decision, approval probability, confidence, risk level, and factor summary.
4. Open **History** to revisit prior submissions.
5. Sign in with the seeded administrator account to view the operations dashboard and model metrics.

The result is a model-assisted demonstration. Users should not interpret a result as a lending commitment.
